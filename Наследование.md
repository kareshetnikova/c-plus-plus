Наследование


Можно без всяких приписок использовать методы 
из класса-родителя.

Пример:
class Animal {
public:
    virtual string getType() = 0;
    virtual bool isDangerous() = 0;
};

class Monkey : public Animal {
public:
    string getType()  { return "Monkey"; }
    bool isDangerous() { return false; }
};
